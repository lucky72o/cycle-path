Design Section 3: UI — Chart Overlays

How the thermal shift interpretation appears on the existing chart

3A. Coverline on the Chart

The coverline is a horizontal line across the chart at the coverline temperature. Its visual style reflects the interpretation state.

Chart Area — Coverline Styles by State

Dashed line, muted purple (#8b5cf6 at 60% opacity). Visible but clearly provisional.
Confirmed (user accepted the interpretation)
36.4°C ⟵ coverline
Solid line, green (#059669). Definitive, user-approved.

engine: 36.4°C
Solid amber line at user's position. Ghost dashed grey line shows engine's original suggestion for reference.

Solid red line with warning indicator. User's confirmed coverline persists but flagged for attention.

The day number row in your existing chart gets subtle background tints. Blue for the 6 reference days, purple for the confirming days, with a thin purple vertical line marking the shift point. Skipped (excluded) days within the reference window could show a striped pattern.

3C. Color System Summary

Interpretation Color Palette
Suggested — dashed purple, muted
Confirmed — solid green
Adjusted — solid amber
Needs Review — solid red + ⚠️
Reference 6 — blue tint on day columns
Confirming 3 — purple tint on day columns
Temp line — existing blue (unchanged)
Fertile window — existing green (OPK, unchanged)
Design principle: Purple (#8b5cf6) is the interpretation color family — it doesn't conflict with existing blue (temps), green (fertile/OPK), pink (intercourse), or grey (excluded). The state colors (green/amber/red) only apply to the coverline itself to signal user decision status.
